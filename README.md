# Exact Golden Spiral Animation

## 概要

p5.jsを使用して、黄金長方形と数学的に正確な対数螺旋（Logarithmic Spiral）のフラクタルアニメーションを描画するWebアプリケーションです。黄金比に基づく長方形の分割、回転、そして螺旋の極（中心）を軸とした無限ズームを視覚的に表現します。

## 主な機能

- **対数螺旋のアニメーション**: 四分円の近似ではなく、極方程式に基づいた正確な対数螺旋を描画します。
- **フラクタルズーム**: 螺旋の中心を軸にして回転・拡大する「Spiral Zoom」、および内側へ吸い込まれるような「Reverse Zoom」機能を搭載しています。
- **インタラクティブなUI**: Tweakpaneを使用したツールウィンドウから、アニメーション速度、サイズ、線の太さなどをリアルタイムに調整できます。
- **カラーとタイポグラフィ**: 10種類のプリセットカラーパレット、配色のランダム化機能、複数のシックなセリフ体フォントの切り替えが可能です。
- **エクスポート機能**:
    - レイアウトを維持した高解像度画像の書き出し。
    - 60fps・高ビットレートでの動画録画（WebM形式）。キーボードショートカット `R` で録画開始、`S` で録画停止が可能です。
    - いずれのエクスポート時にも、当時のパラメータ状態を記録したJSONCファイルが同時に保存されます。
- **設定の保存と復元**: JSONCファイルを読み込んで過去のスケッチ状態を復元できます。
- **履歴管理**: パラメータ操作のUndo（元に戻す）およびRedo（やり直し）に対応しています。

## フェーズ

このアプリケーションでは、無限に続くフラクタル構造を表現するために、アニメーションが大きく3つのフェーズ（コード内では State 1 〜 State 3）に分かれており、これらをループさせています。

1. **State 1（初期表示フェーズ）**
   内容: 1:φの黄金長方形と、数学的に正確な対数螺旋全体が画面に描画されている基準となる状態です。辺の比率を示す数値ラベルも表示されています。

2. **State 2（分割・フェードアウトフェーズ）**
   黄金長方形が「1:1の正方形」と「残りの小さな黄金長方形（1:0.618）」に分割されるプロセスです。視覚的に自然に見せるため、内部的にさらに2つのフェーズに分かれています。
    - 前半フェーズ: 1つの長方形の上から下に向かって、分割線がスッと引かれます。同時に全体の比率ラベル（1.618）が消え、分割後のラベルにクロスフェードします。
    - 後半フェーズ: 切り出された左側の「1:1の正方形」と、その中に含まれる螺旋の一部が徐々に透明になってフェードアウトします。右側の小さな長方形だけが残ります。

3. **State 3（回転・ズームフェーズ）**
   内容: State 2で残った右側の小さな長方形が、対数螺旋の中心（極）を不動の軸として、90度回転しながら拡大します。
   結果: 拡大しきると、元のState 1の巨大な黄金長方形と完全に同じサイズ・向きになり、シームレスにState 1へ戻って無限ループを形成します。

※UIで「Reverse Zoom」をONにした場合は、この時間の流れが逆行し、逆に吸い込まれるようなアニメーションになります。
