# Exact Golden Spiral Animation

## 概要

黄金長方形と黄金螺旋がズームを繰り返すアニメーションを行うWebアプリケーション。p5.jsを使用。

## 機能

- **対数螺旋のアニメーション**:
- **フラクタルズーム**: 螺旋の中心を軸にして回転・拡大する「Spiral Zoom」、および内側へ吸い込まれるような「Reverse Zoom」機能を搭載しています。
- **UI**: ツールウィンドウから、アニメーション速度、サイズ、線の太さなど調整可能・
- **カラーとタイポグラフィ**: 10種類のプリセットカラーパレット、配色のランダム化機能、複数のシックなセリフ体フォントの切り替え可能。
- **エクスポート機能**:
    - レイアウトを維持した高解像度画像の書き出し。
    - 60fps・高ビットレートでの動画録画（WebM形式）。キーボードショートカット `R` で録画開始、`S` で録画停止が可能。
    - いずれのエクスポート時にも、当時のパラメータ状態を記録したJSONCファイルが同時に保存される。
- **設定の保存と復元**: **JSONC** ファイルを読み込んで過去のスケッチ状態を復元できる。
- **履歴管理**: パラメータ操作のUndo/Redoに対応。

## フェーズ

アニメーションが3つのフェーズ（コード内では State 1 〜 State 3）に分かれており、これらをループさせている。

1. **State 1（初期表示フェーズ）**
   内容: 1:φの黄金長方形と、数学的に正確な対数螺旋全体が画面に描画されている状態。辺の比率を示す数値ラベルも表示される。

2. **State 2（分割・フェードアウトフェーズ）**
   黄金長方形が「1:1の正方形」と「残りの小さな黄金長方形（1:0.618）」に分割されるプロセス。視覚的に自然に見せるため、内部的にさらに2つのフェーズに分かれている。
    - 2-1: 1つの長方形の上から下に向かって、分割線が引かれる。同時に全体の比率ラベル（1.618）が消え、分割後のラベルにクロスフェードする。
    - 2-2: 切り出された左側の「1:1の正方形」と、その中に含まれる螺旋の一部が徐々に透明になってフェードアウトする。右側の小さな長方形だけが残る。

3. **State 3（回転・ズームフェーズ）**
   内容: State 2で残った右側の小さな長方形が、対数螺旋の中心（極）を不動の軸として、90度回転しながら拡大する。
   結果: 拡大しきると、元のState 1の巨大な黄金長方形と完全に同じサイズ・向きになり、シームレスにState 1へ戻って無限ループを形成する。

※ UIでReverse ZoomをONにした場合は、逆に吸い込まれるようなアニメーションになる。

## ファイル構造

```text
├── index.html     : アプリケーションのエントリーポイント
├── style.css      : スタイルシート（UIの配置など）
├── main.js        : p5.jsの主要なイベントループ（setup, draw）とアニメーション管理
├── constants.js   : 黄金比やカラーパレットなどの定数定義
├── state.js       : アプリケーション全体の状態管理（appState, animStateなど）
├── draw.js        : キャンバスへの描画処理（ 螺旋計算、State 1〜3の描画）
├── ui.js          : Tweakpane UIの構築とイベント処理、パレット変更ロジック
├── history.js     : Undo/Redo機能と履歴管理
├── export.js      : 画像保存、動画録画、JSONCのロード/セーブ処理
└── utils.js       : 日付フォーマットやイージング、ファイル操作などの汎用関数
```
